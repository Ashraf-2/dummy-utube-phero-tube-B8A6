1. Discuss the scope of var, let, and const?

scope of var: var works like global. var can be declared out of function and can use anywhere of the code. and var can be modified inside and outside of the function and blocks, and within nested loops.
scope of let: the scope of let is actually kind of 'block'.that means if it declared inside of the block, then it can not be accessible out of the block,for example: functions,condtions,and loop _inside of them a let variable can be modified, but outside of them,can not be modified previously declared let variable.
scope of const: this const variable is also a block type scope. work like let. but one important thing is that, const variable can not be changed. const variable can not be modified also.
